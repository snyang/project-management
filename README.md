# 软件项目管理规范

## 介绍

本项目的目的：提供软件项目管理的规范；方便软件技术公司、开发人员使用；减少浪费在管理上的时间。

链接：[软件项目管理规范 on 码云](https://sn-yang.gitee.io/project-management/)
链接：[软件项目管理规范 on Github](https://snyang.github.io/project-management)

许可协议：[MIT](LICENSE)

## 主要特性

- 软件项目管理的方法学
- 软件项目管理的流程
- 软件项目管理的规范
- 软件项目管的模板和示例
- 代码规范
- 一些管理上的闲聊
